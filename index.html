Start with Linux

Linux based operating system are whily used by many type of users because of the benefits that linux brings. First of all, Linux based operating system are pefect for servers, for instance, you can create a web server by using Ubuntu server or or Amazon Linux which as the mane implies it was created based on the Linux Kernel. Second, Linux based operating systems are whily used by developers, because Terminal which a bash Unix shell command line language makes it easy for developers to install and use tools like git, ssh, and the package manager. Finally, the free software community as the name implies most of the software are free to use and depending on the license free to modify.

As I mentioned before, linux based operating system comes in different flavor and here is a table that showes some of the favorite flavors.

Popular Linux Distributions
Ubuntu Kali Linux CenOS Red Hat
Perfect for every day used Perfect for pen-testing Based on a different Kernel Best for servers and mainframes
