Basic Ubuntu Commands

Here I am going to provide a couple of the most basic commands on for Linux Ubuntu, in that way, you can start with the most basic commands.

This first command is the most commond

sudo

Sudo means super user do, this command is one of the most powerful because it will allow you to modify the system, install new programs, and event delete the entire operative system.

Now that we know the most imporant command, we can concentrate into navegating the file system.

cd, ls, mv, cp, rm

First, we are going to start with the cd command which menas change directory, this command allows you to change directories from the command line, then we have the ls which allows you to see the content of the specifit folder , after that we have the mv command which means move and this allows you to move files to different directories. then we have the cp command which you might guess it is means copy and this allows you to copy files. Finally we have the rm command which allows you to remove files form the operative system.

As you can see Linux Ubuntu is not difficult. Thank you for reading.
